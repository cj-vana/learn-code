# Roadmap

## Shipped

The Python curriculum is comprehensive: **1,300+ original items across 70+
topic areas** (910 autograded exercises, 235 lessons, 180 quizzes) organized
into 30 learning paths (7 career, 23 skill).

- **Fundamentals → interview algorithms**: syntax, data structures, debugging,
  and the full pattern catalogue (two pointers, sliding window, prefix sums,
  stacks/queues/BFS, binary search, recursion/backtracking, heaps, intervals &
  greedy, dynamic programming, linked lists, trees, graphs, bits & number
  theory), capped by mixed capstones.
- **Professional Python**: OOP, iterators/generators, decorators/closures,
  exceptions, collections/itertools, regex, files & formats, datetime, testing,
  type hints, and pure-Python data wrangling.
- **AI from scratch** (stdlib only): classic ML, a micrograd-style scalar
  autodiff engine, n-gram language models, BPE tokenization, embeddings &
  retrieval, scaled dot-product attention, and model evaluation.
- **Internals & applied Python**: descriptors & the MRO, the memory model,
  concurrency foundations, modern syntax, functools/operator, binary data &
  struct, numeric precision, introspection, runtime typing, scripting &
  automation, pathlib, CLI parsing, HTTP shapes, SQLite, logging, config
  precedence, functional style, caching & performance, secure coding, and
  dataframe thinking.
- **Capstone projects** end each career path (market ledger, observatory night
  logs, courier dispatch board).
- **Platform**: FastAPI backend, SQLite progress, Typer CLI, gamified
  Vite/React UI, sandboxed runner-broker, adaptive planner with spaced review
  and timed practice, mastery-gated paths with milestones.

## Post-V1 ideas

- More languages beyond Python: JavaScript/TypeScript first, then Go.
- Richer mentor-review flows.
- A hosted multi-user mode (would require an auth/isolation redesign).
